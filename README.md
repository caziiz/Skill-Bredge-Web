# 🎓 SkillBridge

A modern Learning Management System (LMS) built to help students manage their learning journey. SkillBridge allows users to enroll in courses, track progress, manage tasks, and maintain learning streaks to encourage consistent study habits.

---

## 🌟 Overview

SkillBridge is designed to provide a seamless learning experience through an intuitive interface and powerful productivity features. Users can create an account or explore the platform using a demo account, enroll in courses, manage learning tasks, and monitor their progress from a personalized dashboard.

---

## ✨ Features

### 🔐 Authentication
- Secure user registration and login
- Demo account for quick access
- Protected routes
- User session management

### 📚 Course Management
- Browse available courses
- View detailed course information
- Enroll in courses
- Display enrolled courses on the dashboard

### ✅ Task Management
- Create learning tasks
- Mark tasks as completed
- Track pending and completed tasks
- Organize study workflow efficiently

### 🔥 Learning Streak
- Daily login streak tracking
- Motivates users to maintain consistent learning habits
- Automatically updates consecutive login days

### 📊 Dashboard
- Personalized welcome page
- Overview of enrolled courses
- Task statistics
- Completed and unfinished tasks
- Current learning streak

### 📱 Responsive Design
- Mobile-friendly
- Tablet optimized
- Desktop responsive
- Clean and modern UI

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/caziiz/Skill-Bredge-Web.git
```

### Navigate to the project

```bash
cd SkillBridge
```

### Install dependencies

```bash
npm install
```

### Configure environment variables

Create a `.env` file and add the required environment variables.

```env
DATABASE_URL=
JWT_SECRET=
PORT=
```

### Start the development server

```bash
npm run dev
```
